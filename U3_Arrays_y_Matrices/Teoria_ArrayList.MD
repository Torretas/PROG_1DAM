- **¬øQu√© es un `ArrayList`?**

### üìù Teor√≠a Intermedia
**Iteraci√≥n en un ArrayList:** Se puede usar un bucle for, for-each, o forEach con expresiones lambda (Java 8+).
**Tama√±o de un ArrayList:** La funci√≥n size() retorna el n√∫mero de elementos en el ArrayList.

### üíª Ejemplo de C√≥digo

````java

        // Agregar elementos

        frutas.add("Banana");
        frutas.add("Cereza");
        frutas.add("Mango");

        // Imprimir todos los elementos usando un bucle for
        for (int i = 0; i < frutas.size(); i++) {
            System.out.println("Fruta " + (i + 1) + ": " + frutas.get(i));
        }

        System.out.println("::::::::::::::::::::::");

        // Imprimir usando un for-each
        for (String fruta : frutas) {
            System.out.println("Fruta: " + fruta);
        }

        System.out.println("::::::::::::::::::::::");

        // Eliminar un elemento
        frutas.remove("Banana");
        System.out.println("Lista despu√©s de eliminar 'Banana': " + frutas);
    }
}
````
## Explicaci√≥n:

 - **size()**: Muestra el n√∫mero de elementos en el ArrayList.
 - **remove()**: Elimina un elemento por valor o por √≠ndice.
 - Los diferentes bucles (for y for-each) ofrecen distintas formas de iteraci√≥n.

    }

    // M√©todo para eliminar un estudiante

    // M√©todo para mostrar todos los estudiantes

            }
        }
    }
}
