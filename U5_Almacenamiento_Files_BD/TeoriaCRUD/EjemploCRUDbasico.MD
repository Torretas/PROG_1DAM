## üß† Clase `UsuarioDAOImpl.java`
```java
package dao;

import model.Usuario;
import util.ConexionBD;

import java.sql.*;
import java.util.ArrayList;
import java.util.List;

public class UsuarioDAOImpl implements UsuarioDAO {

    public void crear(Usuario usuario) {
        String sql = "INSERT INTO usuarios (nombre, email) VALUES (?, ?)";
        try (Connection conn = ConexionBD.getConexion();
             PreparedStatement stmt = conn.prepareStatement(sql)) {
            stmt.setString(1, usuario.getNombre());
            stmt.setString(2, usuario.getEmail());
            stmt.executeUpdate();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    public List<Usuario> obtenerTodos() {
        List<Usuario> lista = new ArrayList<>();
        String sql = "SELECT * FROM usuarios";
        try (Connection conn = ConexionBD.getConexion();
             Statement stmt = conn.createStatement();
             ResultSet rs = stmt.executeQuery(sql)) {
            while (rs.next()) {
                Usuario u = new Usuario(
                    rs.getInt("id"),
                    rs.getString("nombre"),
                    rs.getString("email")
                );
                lista.add(u);
            }
        } catch (SQLException e) {
            e.printStackTrace();
        }
        return lista;
    }

    public void actualizar(Usuario usuario) {
        String sql = "UPDATE usuarios SET nombre = ?, email = ? WHERE id = ?";
        try (Connection conn = ConexionBD.getConexion();
             PreparedStatement stmt = conn.prepareStatement(sql)) {
            stmt.setString(1, usuario.getNombre());
            stmt.setString(2, usuario.getEmail());
            stmt.setInt(3, usuario.getId());
            stmt.executeUpdate();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }

    public void eliminar(int id) {
        String sql = "DELETE FROM usuarios WHERE id = ?";
        try (Connection conn = ConexionBD.getConexion();
             PreparedStatement stmt = conn.prepareStatement(sql)) {
            stmt.setInt(1, id);
            stmt.executeUpdate();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
```

# üí° Papel de JSON en un CRUD en Java

---

## 1. üì¶ Intercambio de datos (cliente-servidor)

En aplicaciones web modernas, los datos que se env√≠an desde el **frontend (Angular, React, etc.)** al **backend Java** suelen ir en formato **JSON**.

### Ejemplo

Un formulario env√≠a al backend este objeto JSON:
```json
{
"nombre": "Ana Torres",
"email": "ana@example.com"
}
```
El backend en Java lo recibe, lo convierte a un objeto `Usuario`, lo guarda en la base de datos (operaci√≥n **Create**) y responde con otro JSON.

---

## 2. üß© Serializaci√≥n y deserializaci√≥n en Java

En Java puedes usar bibliotecas como **Jackson** para convertir entre objetos y JSON (serializar/deserializar).

### Ejemplo con Jackson
```java
ObjectMapper mapper = new ObjectMapper();

String json = "{\"nombre\":\"Pedro\",\"email\":\"pedro@correo.com\"}";

Usuario usuario = mapper.readValue(json, Usuario.class);  // Deserializa JSON a objeto

String jsonGenerado = mapper.writeValueAsString(usuario);  // Serializa objeto a JSON
```
---

## 3. üíæ Persistencia alternativa con JSON

En lugar de usar una base de datos, puedes almacenar los datos de la aplicaci√≥n en archivos `.json`. Esto permite:

- Prototipos o pruebas sin montar un SGBD.
- Persistencia b√°sica sin instalaci√≥n de MySQL.

### Ejemplo de archivo `usuarios.json`
```json
[
{
"id": 1,
"nombre": "Juan",
"email": "juan@mail.com"
},
{
"id": 2,
"nombre": "Luc√≠a",
"email": "lucia@mail.com"

```
### Guardar un objeto Java en JSON
```java

mapper.writeValue(new File("usuarios.json"), listaUsuarios);
```
### Leer desde JSON y cargar en lista
```java

new TypeReference<List<Usuario>>() {});
```
